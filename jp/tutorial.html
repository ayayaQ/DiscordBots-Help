基本

ボットのセッション間で保存したい情報がある場合、"botState"という変数を利用できます。使い方は非常に簡単です。

  1. コマンド作成画面でJavaScriptを使用するコマンドを作成してください。方法が分からない場合は「メッセージでJavaScriptを使う」チュートリアルを参照してください。
  2. 情報を保存したい場合はbotState変数にプロパティを設定します。例：botState.x = '保存した文字列'
  3. 保存した情報をメッセージに挿入するには、$get(botState.x)を使用します。これで「保存した文字列」が表示されます。
  4. これだけでOKです。アプリはホスティング終了時やアプリ終了時にbotStateを自動保存します。

このコマンドはD&Dなどで使えるサイコロの出目をシミュレートします。

  1. コマンド作成画面でタイプを「メッセージ受信」に設定します。
  2. コマンド名は「!dice」とします。
  3. チャンネルメッセージ欄に「You rolled a $rollnum(1,6)」と入力します。

このコマンドはキック権限を持つユーザーが他ユーザーをキックできます。

  1. コマンド作成画面でタイプを「メッセージ受信」に設定します。
  2. コマンド名は「!kick」とします。
  3. モデレーション欄で「kick」にチェックを入れます。
  4. コマンドを保存し、ボットをホストしてテストしてください。使用例：「!kick @user」

このコマンドはユーザーがサーバーに参加した際にロールを付与します。

  1. コマンド作成画面でタイプを「メンバー参加」に設定します。
  2. 「Assign/Remove Role」にチェックを入れます。初回はロール付与、再度はロール削除となります。「メンバー参加」タイプでは一度のみ発動します。
  3. 「Role Name」欄にロール名を入力します（例：New user）。大文字小文字は区別されます。
  4. 誰かがサーバーに参加した際にコマンドをテストしてください。

コマンド例

埋め込みメッセージは特別なタイプのメッセージです。作成方法を説明します。

  1. コマンド作成画面でタイプを「メッセージ受信」、コマンド名を「!embed」に設定します。
  2. 「Send Channel Message Embed」にチェックを入れます。
  3. 5つのフィールドが表示されます。最低1つ入力すればOKです。
  4. タイトル欄に「My Embed」と入力します。
  5. 説明欄に「$name is who called this command」と入力します。
  6. フッター欄に「$date」と入力します（タイムスタンプとして利用可能）。
  7. 画像URL欄に「https://i.redd.it/9hs5qf7qulz41.png」と入力します。任意の画像リンクを指定可能です。
  8. サムネイル欄に「$avatar」と入力します。コマンド呼び出し者のアバターURLが入ります。任意の画像リンクも指定可能です。
  9. コマンドを保存し、テストしてください。

このコマンドは特定メッセージにリアクションしてロールを付与・削除します。

  1. タイプを「リアクション追加」でコマンドを作成します。
  2. コマンド欄にロール付与用の絵文字を入力します（例：😊）。カスタム絵文字は名前のみ指定してください。
  3. 「Require specific message ID」にチェックを入れ、対象メッセージIDを入力します。
  4. 「Assign/Remove Role」にチェックを入れ、付与するロール名を入力します（例：Tester）。
  5. コマンドをテストし、他のロールも同様に設定できます。

コマンド例

JavaScriptを使うには$evalや$haltなどのキーワードを利用します。

  1. 任意の名前でコマンドを作成します（例：「Hello world」）。
  2. チャンネル送信にチェックを入れます。
  3. $evalキーワードと$haltキーワードを挿入します。
  4. 間にコードを記述します（例：「var hello = "Hello world";」）。
  5. $haltの後または$evalの前に残りのメッセージを記述します（例：「Our output is $get(hello)」）。
  6. ボットを実行し、コマンドをテストしてください。
